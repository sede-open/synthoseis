# Output format

All volumes are written as **Zarr v3** stores — directory trees of chunked,
compressed arrays.

## Directory layout

```
project_folder/
└── <project>/
    └── <run_id>/
        ├── seismic_angle_07.zarr/
        ├── seismic_angle_15.zarr/
        ├── seismic_angle_24.zarr/
        ├── geologic_age.zarr/
        ├── fault_segments.zarr/
        ├── net_to_gross.zarr/
        ├── oil_closures.zarr/
        ├── gas_closures.zarr/
        ├── brine_closures.zarr/
        ├── salt_body.zarr/      (if include_salt=true)
        └── config_<run_id>.json
```

Each `.zarr` is a standalone store — no shared metadata between volumes.

## Array properties

| Property | Value |
|----------|-------|
| Shape | `cube_shape` → `[X, Y, Z]` |
| dtype | `float32` |
| Chunk layout | Auto-selected by Zarr (typically `[64, 64, 64]`) |

## Reading with Python

```python
import zarr
import numpy as np

store = zarr.open("path/to/seismic_angle_07.zarr", mode="r")
data = store[:]          # load full array → np.ndarray (float32)
inline = store[150, :, :]  # single inline slice
```

## Reading with xarray

```python
import xarray as xr

ds = xr.open_zarr("path/to/seismic_angle_07.zarr")
```

## Manifest

The `manifest.json` file at the root of `project_folder` is generated by
`scripts/generate_manifest.py` and lists every completed run and its available
volumes. The dashboard and `GET /api/manifest` both consume this file.

```json
[
  {
    "run_id":  "my_first_run",
    "path":    "/abs/path/to/my_first_run",
    "volumes": ["seismic_angle_07", "seismic_angle_15", "geologic_age", "..."]
  }
]
```

## Why Zarr?

- **Cloud-native**: Zarr stores can be written to and read from S3, GCS, or
  Azure Blob Storage with no format changes.
- **Lazy loading**: Consumers load only the slices they need.
- **No HDF5 dependency**: Earlier versions wrote HDF5; Zarr removes the
  single-writer lock and the libhdf5 C dependency.
- **Interoperable**: Readable by Python (`zarr`), Julia (`Zarr.jl`), and any
  language with a Zarr v3 implementation.
